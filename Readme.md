# HerChain AI 🌸

[![Vite](https://img.shields.io/badge/Frontend-Vite%20%2B%20React-646CFF?style=for-the-badge&logo=vite&logoColor=white)](https://vitejs.dev/)

[![Blockchain](https://img.shields.io/badge/Security-Blockchain-363636?style=for-the-badge&logo=blockchain&logoColor=white)](https://blockchain.com/)

**HerChain AI** is a premium, AI-powered women's healthcare platform designed to provide personalized wellness insights, secure health data management via blockchain, and intelligent agentic support.

---

## ✨ Key Features

-   **🤖 AI Health Assistant**: Intelligent chat interface powered by specialized agents for personalized health guidance.
-   **📊 ML Predictions**: Advanced machine learning models for health forecasting and wellness trends.
-   **🔗 Blockchain Security**: Decentralized verification and secure storage for sensitive health records.
-   **📱 Modern Dashboard**: Real-time tracking of wellness metrics, cycle health, and personalized analytics.
-   **🎨 Premium UI/UX**: A sleek, responsive, and emotionally supportive interface built with modern web technologies.

---

## 🛠️ Tech Stack

### Frontend
- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Vanilla CSS / Tailwind CSS (Optimized for premium aesthetics)
- **State Management**: React Hooks & Context API

### Backend
- **Framework**: FastAPI (Python)
- **AI/ML**: Agentic workflows and ML prediction models
- **Security**: Blockchain-based verification system
- **Environment**: Python 3.9+

## 🚀 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [Python](https://www.python.org/) (v3.9+)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Biraj021/Herchain_AI.git
   cd Herchain_AI
   ```

2. **Setup Backend**
   ```bash
   cd backend
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   python main.py
   ```

3. **Setup Frontend**
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
